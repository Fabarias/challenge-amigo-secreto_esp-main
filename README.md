# 🕵️ Proyecto: Amigo Secreto

  App web que te permite sortear una lista de amigos, donde te dará el amigo secreto de manera aleatoria.

## 🧐 Características

  - Ingresar una lista de amigos por sus nombres, uno por uno.
  - Sortear un amigo secreto.
  - Mostrar el nombre del amigo sorteado.
  - Capacidad de reiniciar lista.

## 🛠️ Herramientas utilizadas

  - HTML
  - CSS
  - JavaScript

## 📋 Condiciones de uso

  1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/amigo-secreto.git
   ```
  2. Abre `index.html` en tu navegador de preferencia.

     También puedes usarlo en el siguiente enlace:

      a. https://amigo-secreto-gbldzm1b4-fabricio-arias-projects-74046134.vercel.app

## 📁 Estructura del proyecto

  - `index.html`: Cuerpo principal de la página.
  - `style.css`: Estilos.
  - `app.js`: Lógica y función del programa.
